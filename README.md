# EMSC2010 – Week 4 Practical 2: Plotting the Keeling Curve

This repository contains the template Jupyter notebook for **Week 4 Practical 2** of *EMSC2010: Data Science for Earth System Scientists* at the Australian National University.

The session is a self-directed exercise in **finding, importing, and plotting real-world data**, using the Mauna Loa atmospheric CO₂ record (the *Keeling curve*) as the subject.

---

## Notebook

### Notebook 1 – Plotting the Keeling Curve (`NB1`)

**Dataset:** Monthly mean atmospheric CO₂ concentration measured at Mauna Loa Observatory, Hawaii — the longest continuous direct record of atmospheric CO₂, named the *Keeling curve* after Charles Keeling, who established and ran the monitoring program until his death in 2005.

This is an open-ended, self-directed exercise. Rather than working through pre-written code, students are asked to:

1. Search for and download the Mauna Loa monthly CO₂ data, saving it as an Excel workbook (or similar)
2. Create a **new** Colab notebook (no GitHub template required for this step)
3. Mount and read the data into the notebook using `pandas`
4. Plot CO₂ concentration against time using `matplotlib`, with appropriately labelled axes
5. Annotate the notebook to describe the workflow, explain the code, and cite data sources

This exercise consolidates the data import and plotting skills introduced in Weeks 3–4, applied to sourcing and working with a real dataset independently.

**Key concepts:** Sourcing real-world data, data import and mounting in Colab, `pandas` data loading, `matplotlib` time series plotting, documentation and citation of data sources

**Libraries:** `pandas`, `matplotlib`

> **Note:** This exercise is explicitly stated as **not** part of Assessment 1.

2. Open the notebook from your copy of the repository and click the **"Open in Colab"** badge at the top of the notebook to launch it in Google Colab.

4. For the Keeling curve exercise itself, follow the in-notebook instructions — note that this part of the exercise can be completed in a **new** Colab notebook rather than this template.
